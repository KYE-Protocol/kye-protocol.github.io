v1.0.0 · April 2026

First public release.

The frozen v1.0 contract: 36 v1.0 normative profiles + 5 v1.1 preview, 193 OpenAPI operations across 87+ runtime endpoints, 57 JSON Schemas with 60 validated examples, the reference Gateway, embedded PDP, PEP middleware, three SDKs (TypeScript / Python / Go), 38 conformance fixtures (38/38 pass), 266 compliance control mappings.

  • Apache License 2.0 across the protocol-adoption layer.
  • KYE Compliance Mapping Rail™ shipped with control bindings for SOC 2, ISO 27001:2022, PCI DSS 4.0, PSD2/PSD3, DORA, NIS2, EU AI Act, NIST 800-207, HIPAA.
  • Patent-safety scan (mechanism-leak block) wired into CI; 0 violations across all 6 public org repos.
  • KYE™ Self-Audit & Attestation Profile™ normative spec; 7 SQL tables; 9 schemas; 9 examples; 74 OpenAPI operations under /v1/conformance/*, /v1/certification/*, /v1/self-audit/*.
